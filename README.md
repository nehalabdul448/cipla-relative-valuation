Cipla Relative Valuation Model
This project performs a relative valuation of Cipla Ltd. using comparable company analysis.
Data source: Screener.in
Methods used
P/E multiple
EV/EBITDA
EV/Revenue

Process
Collected financial data of peer companies
Calculated average multiples
Estimated implied equity value
Compared implied share price with current market price

Tools
Microsoft Excel
Financial data from Screener.in
